AI Placement Assistant
An intelligent web application designed to help students prepare for campus placements. This tool provides guidance on interview preparation, resume building, and company-specific coding practice.

Features
Interview Prep: Strategic guidance for common interview scenarios.

Resume Help: Tips and structures for building high-impact resumes.

Company Guidance: Insights into company-specific hiring patterns.

Coding Practice: Exercises to sharpen technical skills.

Tech Stack
Frontend: HTML, CSS

Logic/Backend: JavaScript (app.js)

Library: Antigravity

Data: Custom dataset (dataset.txt) used for training the assistant's responses.

Project Structure
index.html: The main user interface.

style.css: Custom styling for a professional look.

app.js: Core logic powered by Antigravity.

dataset.txt: The knowledge base for the AI.
